## Install

Via Composer
``` bash
$ composer require hamidjavadi/guid
```


##Usage
``` php

<?php

use Hamidjavadi\guid;

echo guid::generate();
// 


$guid = new guid();

echo $guid->generate();
//

echo $guid->generate();
//
```
